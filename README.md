## RXJS Operators Demo

1. Clone the repo
2. Open the created folder
3. run `npm ci`
4. run `npm start`

This repo is meant to be a progression through the commits, not a standalone example of each operator.

Documentation for all operators:

https://www.learnrxjs.io/learn-rxjs/operators
